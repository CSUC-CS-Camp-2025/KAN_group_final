Yahtzee


How to Play

Yahtzee is a dice game where the goal is to maximize your score as much as possible. To do this, you must use strategy since you can only input one score at a time. The scoring will count whatever dice you have saved, and none of the dice on the board.

Scoring Catgeories

Aces, Twos, Threes, Fours, Fives, Sixes: These categories simply add however many of the mentioned dice are in the dice you have selected, and awards you the points of the dice total.

Bonus: This category relies on the total score of your upper section categories. You need 63 or more points across the upper section to earn this catgory, rewarding you 35 points.

3 of a Kind: This category will only score if you have 3 dice of the same number selected. If you do, then it will add all your dice you have selected, and award that amount of points.

4 of a Kind: This category will only score if you have 4 dice of the same number selected. If you do, then it will add all your dice you have selected, and award that amount of points.

Full House: This category will only score if you have 2 dice of the same kind, and 3 dice of the same kind. For example, if you rolled (3, 3, 3, 5, 5). This awards 25 points.

Small Straight: This category scores if you have a sequence of 4 dice. For example, (1, 2, 3, 4). This awards 30 points.

Large Straight: This category scores if you have a sequence of 5 dice. For example, (1, 2, 3, 4, 5). This awards 40 points.

Yahtzee: This category scores if you have 5 dice of the same value. This awards a whopping 50 points!

Chance: This category awards the sum of all the selected dice. For example, (1, 3, 5, 6, 2) would award 17 points. Tip: this catgeory is useful if you are in a pinch.
